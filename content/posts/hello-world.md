## 你好，世界

这是 Jansen 的技术博客，基于 **Hugo** + **GitHub Pages** 搭建。

### 搭建流程回顾

1. 安装 Hugo（`brew install hugo`）
2. 初始化站点（`hugo new site`）
3. 安装 PaperMod 主题
4. 配置 `hugo.toml`（中文、菜单、元信息）
5. 配置 GitHub Actions 自动部署到 `aha-jansen.github.io`

### 为什么选择 Hugo？

- **Go 技术栈一致**：Hugo 是 Go 写的，模板语法对 Go 开发者非常直观
- **构建极快**：数千篇文章也能亚秒级完成构建
- **零依赖**：单二进制文件，`brew install` 即用
