# Jansen's Blog

基于 [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 搭建的个人博客。

## 在线访问

🔗 [https://aha-jansen.github.io/](https://aha-jansen.github.io/)

## 在线编辑

可通过 github.dev 在浏览器中直接编辑，无需本地环境：

👉 [https://github.dev/aha-jansen/aha-jansen.github.io](https://github.dev/aha-jansen/aha-jansen.github.io)

- 按 `.` (句号键) 直接在 GitHub 仓库页面进入编辑模式
- 或手动将 `github.com` 替换为 `github.dev`

## 本地运行

```bash
# 安装 Hugo (macOS)
brew install hugo

# 启动开发服务器
hugo server -D

# 访问 http://localhost:1313
```
